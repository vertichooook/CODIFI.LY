## 📖 О проекте

**Codifi.ly** — это образовательная веб-платформа, которая позволяет изучать программирование на Python прямо в браузере. Процесс обучения разбит на удобные шаги: от изучения теории и прохождения квизов до написания реального кода с мгновенной проверкой. 

Никаких скучных лекций. Только практика, геймификация и современный дизайн!

## ✨ Главные фичи

- 💻 **Встроенная IDE (Skulpt)** — пишите и запускайте Python-код прямо в браузере без установки локального окружения.
- 🎯 **Система прогресса** — зарабатывайте XP, повышайте ранг и отслеживайте свои успехи.
- 🌓 **Умный дизайн** — поддержка светлой и темной (Glassmorphism) темы оформления.
- 🌍 **Мультиязычность** — интерфейс доступен на русском, английском и немецком языках.
- 🔍 **Умный справочник** — быстрый поиск по всей теории курса без отрыва от решения задач.
- 🎉 **Геймификация** — визуальный отклик (конфетти и звуки) при правильном решении задач.
